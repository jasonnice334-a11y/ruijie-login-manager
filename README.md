# ⚡ ALL RUIJIE HACK ⚡

> **Telegram: @naymin126653**

Ruijie (锐捷) WiFi portal မှ voucher code သုံးပြီး internet ချိတ်ဆက်ရန် tool ဖြစ်သည်။

---

## 📋 လိုအပ်သောအရာများ

- Android ဖုန်း + Termux app
- Ruijie WiFi network ချိတ်ဆက်ထားရမည်
- Valid voucher code

---

## ⚙️ Install လုပ်နည်း (Termux)

```bash
pkg update && pkg upgrade -y
pkg install python git -y
pip install aiohttp requests
```

---

## 📥 Download နည်း

```bash
git clone https://github.com/jasonnice334-a11y/ruijie-login-manager
cd ruijie-login-manager
```

---

## ▶️ Run နည်း

```bash
python nay.py
```

Run ပြီးရင် voucher code ထည့်ပါ။

---

## ⚠️ မှတ်ချက်

- Ruijie network ကို ချိတ်ဆက်ထားသော WiFi ပေါ်မှ run မှသာ အလုပ်လုပ်မည်
- Gateway IP နှင့် MAC address ကို အလိုအလျောက် detect လုပ်သည်
